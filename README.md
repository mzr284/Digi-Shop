# Digi Shop
A shopping web application built with **react** and **node.js**. Data of products featch from an API.
You can added product to cart, increment count of products, remove these from your cart, or buy all of products in your cart. If you are admin you can enter to admin dashboard
and see all of users and products and you can edit these. Also you can see all of purchases that created when a user buy some products.
Also this site has very good responsive design for laptob, tablet and mobile.
